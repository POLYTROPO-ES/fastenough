# Fast enough

Small PWA to see the correlation between speed and time saved.

## What the App Does
- Displays a car-style speedometer focused on travel speed and time impact.
- Supports two user-controlled speed values:
  - Current speed
  - Speed limit
- Supports trip distance from 10 to 1000 with an in-dial vertical slider.
- Calculates and shows time difference for typical speed marks.
- Supports kilometers and miles.
- Supports multiple UI languages.

## Current UX Model
- Main visual is the speedometer.
- Speedometer includes clickable/touchable rings:
  - Limit ring
  - Current speed ring
- Ring colors include legend guidance in the top-right corner.
- Speed limit indicator uses a limit-sign style marker (red circle with white center) plus a short limit tick.
- Current speed indicator uses main needle with a short tail behind center.

## Tech Stack
- React 19
- TypeScript 5
- Vite 8
- Vitest + Testing Library
- Playwright
- Cloudflare Pages + Pages Functions
- ESLint + Prettier

## PWA
- Manifest: public/manifest.webmanifest
- Service worker: public/sw.js
- Registration: src/main.tsx
- App metadata: index.html

## Scripts
- npm run dev: local development server
- npm run build: typecheck + production build
- npm run preview: preview production build
- npm run typecheck: TypeScript checks
- npm run lint: ESLint checks
- npm run format: Prettier formatting
- npm run test: Vitest watch mode
- npm run test:run: Vitest run + coverage
- npm run test:e2e: Playwright tests
- npm run cf:deploy: deploy dist to Cloudflare Pages
- npm run cf:dev: local Cloudflare Pages preview

## Testing
- Unit/component tests live in src and src/lib.
- E2E tests live in tests/e2e.
- Coverage is generated by npm run test:run.

## Project Structure
- src: app UI, speedometer logic, styles
- src/lib: helper logic
- src/test: test setup
- tests/e2e: end-to-end tests
- functions: Cloudflare Pages Functions
- public: static files, headers, redirects, manifest, service worker
- docs: product goals, deployment docs, decision records

## Deployment Notes
- Cloudflare project defaults to fastenough.
- Security headers are configured in public/_headers.
- SPA redirects are configured in public/_redirects.
